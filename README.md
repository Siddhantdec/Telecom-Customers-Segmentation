# Telecom-Customers-Segmentation
Customer segmentation is the division of potential customers in a given market into discrete groups. We would be defining marketing strategy by analyzing their customer's behavior.
We have applied a data reduction technique factor analysis for variable reduction and we have used a KNN Clustering algorithm to reveal the behavioral segments.
We have also identified the cluster characteristics of the cluster using detailed profiling.
Providing the strategic insights and the implementation of strategies for given set of cluster characteristics.
